XV's Password Generator

A simple example creating a password generator using Vue.js v2.x.


Script Demo


How It Works

Characters
Is Optional
The characters attribute is used to set the character type used in the password. You can use numbers 0-9, letters a-z (and or) A-Z, and spical characters #. All of these sets can be used individually, but they all work together too.

Size
Is Optional
The size attribute is used to set the number of characters that is used in the field, if you need a password with 12 characters, then set this attribute to 12.

Type
Is Optional
The type attribute is used to change the field type from text to password.

Placeholder
Is Optional
The placeholder attribute is used to set the place holder text on the field input.

Auto
Is Optional
The auto attribute is used to suppress the automatic generation of the password on load, if you don't want the field to automatically generate the password on load, then pass false to the component.
